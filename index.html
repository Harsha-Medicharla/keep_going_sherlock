Sherlock's Puzzle: The Valuable Scroll

Sherlock Holmes is once again in a tricky situation. This time, he needs to organize a series of clues, each represented by a word. These clues must be written on two scrolls. The first scroll has a fixed length, while the second scroll is infinitely long. Sherlock's goal is to maximize the number of clues that can be written on the first scroll, ensuring that no clue is split between the two scrolls.

The first scroll has limited capacity, and Sherlock must figure out how many clues can fit on it before it exceeds the given capacity. The remaining clues are then written on the second scroll, which can hold an unlimited number of characters.

Problem Description

Sherlock has a list of clues represented by n words. The first scroll has a limited capacity of m characters. Sherlock must determine the maximum number of clues (x) that can fit on the first scroll such that the total length of the first x words is no greater than m. Each word must be written in full on the first scroll; no word can be split between the two scrolls.

Your task is to find the maximum number x such that the total length of the first x clues fits within the first scroll's capacity m.

Input

The first line contains an integer t (1 ≤ t ≤ 1000) — the number of test cases.

For each test case:

  • The first line contains two integers n and m (1 ≤ n ≤ 50; 1 ≤ m ≤ 500) — the number of clues (words) and the maximum number of characters that can fit on the first scroll.
  • The next n lines each contain one word s_i (1 ≤ |s_i| ≤ 10) made up of lowercase Latin letters.

Output

For each test case, output the maximum number x such that the first x clues fit on the first scroll without exceeding the capacity m.

The first three clues "three" (5), "eight" (5), and "the" (3) have a total length of 13, which fits within the 15-character limit. The next clue "man" (3 characters) would exceed the 15-character capacity. Therefore, the answer is 3.
  • Test case 2: The first scroll has a capacity of 10 characters. The clues are: ["serious", "the", "case", "is"]. The first two clues "serious" (7 characters) and "the" (3 characters) together take up 10 characters, which is exactly the capacity of the first scroll. The next clue "case" (4 characters) exceeds the limit. Therefore, the answer is 2.
  • Test case 3: The first scroll has a capacity of 8 characters. The clues are: ["he", "good"]. The first clue "he" (2 characters) and "good" (4 characters) together fit within the 8-character limit. Therefore, the answer is 2.
